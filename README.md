# ✝️ Panis Angelicus | Milagres Eucarísticos

Este projeto é uma plataforma digital dedicada à catalogação e divulgação dos **Milagres Eucarísticos** documentados pela Igreja Católica ao longo dos séculos. O objetivo central é oferecer uma interface solene e imersiva para o estudo desses eventos, servindo como uma biblioteca digital acessível e responsiva.

## 🚀 Tecnologias e Implementação

O desenvolvimento focou em performance, semântica e uma experiência de usuário (UX) focada na leitura:

* **Arquitetura:** HTML5 Semântico para SEO e estrutura de dados clara.
* **Estilização:** CSS3 utilizando *Custom Properties* (Variáveis) para gestão de temas e *Media Queries* avançadas para total responsividade.
* **Interatividade:** JavaScript Vanilla para manipulação de eventos de scroll (Navbar dinâmica), controle do menu mobile e animações de entrada (Fade-in).
* **Gráficos:** Vetores SVG para ícones e favicon, garantindo leveza e nitidez em qualquer densidade de pixels.

## 📱 Diferenciais Técnicos

* **Layout Editorial:** Focado na tipografia clássica e legibilidade de textos longos.
* **Mobile-First:** Menu hambúrguer customizado e grid adaptável para smartphones.
* **Performance:** Código limpo, sem dependências de frameworks externos ou bibliotecas pesadas.
